HTML PAGE STRUCTRUE

HTML (Hypertext Markup Language) is the foundation of the World Wide Web, and its page structure is the framework upon which web content is built. The structure of an HTML page is essential for organizing and presenting content on the internet. Here's a comprehensive description of the HTML page structure

CODE:

        
    <!--This declaration tells the browser that this document is an HTML file -->
    <!DOCTYPE html>

    <!--The <html> tag is the root element of an HTML document. All HTML content should be placed inside this tag. -->
    <html>

        <!--The <head> element contains metadata about the document, such as character encoding and the document's title. -->
        <head>
            <!--This meta tag specifies the character encoding as UTF-8, which supports a wide range of characters and symbols. -->
            <meta charset="UTF-8">

            <!--This meta tag sets the initial viewport properties for responsive design. It ensures the webpage's width matches the device's width. -->
            <meta name="viewport" content="width=device-width, initial-scale=1.0">

            <!--The <title> tag defines the title of the webpage, which appears in the browser tab. -->
            <title>WEB 1 LAB TASKS</title>
        </head>

        <!--The <body> element contains the visible content of the webpage. Everything you want to display on the page goes inside this tag. -->
        <body>
        <!--You can add content here, such as text, images, links, and more. -->
        </body>
        
        <!--The closing </html> tag marks the end -->
    </html>
